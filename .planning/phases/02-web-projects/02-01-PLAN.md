**Create web/js/main.js** — Minimal JavaScript for interactivity:

1. **Mobile nav toggle:**
   - Select `.nav-toggle` button and `.nav-mobile` overlay
   - On toggle click: toggle `.active` class on nav-mobile, update aria-expanded attribute
   - On nav-mobile link click: close menu, update aria-expanded
   - On Escape key: close menu if open

2. **Smooth scroll for anchor links:**
   - Select all `a[href^="#"]` links
   - On click: preventDefault, use `element.scrollIntoView({ behavior: 'smooth', block: 'start' })`
   - Account for sticky nav height offset (get nav height, subtract from scroll position)

3. **Active nav link highlighting:**
   - Use IntersectionObserver on sections
   - When section enters viewport (threshold 0.3), add `.active` class to corresponding nav link
   - Remove `.active` from other nav links

**Critical rules:**
- Zero primitive token references in main.css — only semantic tokens (--color-*, --spacing-*, --font-*, --border-radius-*, --line-height-*)
- All var() calls include fallback values
- Use :focus-visible (not :focus) for non-intrusive focus styles
- JavaScript is progressive enhancement — page must work without JS (links still navigate via anchor)
