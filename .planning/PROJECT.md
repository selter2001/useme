# UsemePortfolio — Profil Freelancera i Portfolio

## What This Is

Kompletny pakiet startowy dla freelancera na platformie Useme.com — zoptymalizowany profil + 4 w pełni działające projekty portfolio. Wojciech Olszak specjalizuje się w szybkim tworzeniu aplikacji iOS (SwiftUI) oraz stron internetowych (HTML/CSS/JS), wykorzystując AI (Claude Code) jako kluczowy wyróżnik. Projekty hostowane na GitHub Pages, apki iOS dostępne jako kod Xcode + web-demo.

## Core Value

Wiarygodne, profesjonalne portfolio z działającym kodem, które przekona potencjalnych klientów Useme do zlecenia pierwszego projektu.

### Out of Scope

- Backend/bazy danych — projekty działają lokalnie lub jako statyczne strony
- Pełna integracja chatbota AI — tylko UI mockup jeśli potrzebny
- React/Vue/Next.js — czyste HTML/CSS/JS, najprostszy stack dla GitHub Pages
- App Store deployment — kod Xcode do uruchomienia lokalnie, nie do publikacji
- Automatyczne testy — projekty są demonstracyjne

## Context

**Sytuacja:** Wojciech ma doświadczenie z własnymi projektami — m.in. aplikacja iOS do wyliczania oprysku potrzebnego do opryskiwania pola (zlecenie od brata). Nie ma jeszcze komercyjnych zleceń przez Useme. Potrzebuje profesjonalnego profilu i portfolio, żeby zacząć zdobywać klientów.

**Platforma docelowa:** Useme.com — polski marketplace freelancerski. Profil musi być po polsku, opisy projektów i kod dwujęzycznie (PL/EN).

**Styl wizualny:** Ciemny/Tech — dark mode, neonowe akcenty, minimalistyczny styl "dev". Spójny w całym portfolio.

**Hosting:** GitHub Pages — statyczne strony, zero konfiguracji serwera.

**iOS projekty:** Podwójny format — prawdziwy kod SwiftUI (projekt Xcode) + web-demo w HTML/CSS/JS symulujący wygląd apki iOS (do oglądania bez Xcode).

**Wyróżnik profilu:** AI-assisted development z Claude Code — szybkość, jakość, nowoczesne podejście.

## Constraints

- **Stack webowy**: Czysty HTML/CSS/JS — brak frameworków, GitHub Pages kompatybilny
- **Stack iOS**: SwiftUI — nowoczesny, deklaratywny framework Apple
- **Język profilu**: Polski opis, dwujęzyczne projekty
- **Design**: Dark/Tech theme, spójny w całym portfolio
- **Hosting**: GitHub Pages — darmowe, proste
